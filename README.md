# 🛍️ AI39C-Group6 Single Vendor E-Commerce

> A complete e-commerce platform built with Flask and MySQL for single vendor operations.

---

## 📌 About The Project

This project is a **Single Vendor E-Commerce** web application developed as part of the AI39C Group 6 assignment. It allows an admin to manage products and categories efficiently with a clean and modern interface.

---

## ✨ Features

### 🔐 Authentication
- Admin login system
- Secure session management

### 📁 Category Management
- Add new categories with images
- Edit existing categories
- Delete categories
- Create sub-categories

### 📦 Product Management
- Add products with images
- Edit product details
- Delete products
- Assign products to categories

### 🎨 User Interface
- Red, White & Blue theme
- Responsive design
- Clean and modern layout

## 🚀 Getting Started

### Prerequisites
- Python 3.8+
- MySQL Server

### Installation

**1. Clone the repository**
```bash
git clone https://github.com/yagyarazz77/AI39C-Group6_project-SingleVendor_E-Commerce.git
cd AI39C-Group6_project-SingleVendor_E-Commerce
```

**2. Create and activate virtual environment**
```bash
python3 -m venv .venv
source .venv/bin/activate  # Mac/Linux
# OR
.venv\Scriptsctivate     # Windows
```

**3. Install dependencies**
```bash
pip install -r requirements.txt
```

**4. Setup database**
```sql
CREATE DATABASE class_db;
```

**5. Run the application**
```bash
python3 run.py
```

**6. Access the application**
```
http://127.0.0.1:5003
```
